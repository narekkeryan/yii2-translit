Armenian - Latin Translit
================

Yii2 Translit for Armenian Language

Installation
============

```
composer require "narekkeryan/yii2-translit":"dev-master"
```

Usage example
=============

```php
use keryan\translit\Translit;

Translit::translit('Բարև աշխարհ');
```
